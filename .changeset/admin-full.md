The admin gains a public `/api/admin_providers` endpoint listing configured OAuth and SSO provider ids for the login page's sign-in buttons; a `/users` page for viewing and updating user roles via the better-auth admin plugin; a `/revisions` page for browsing snapshot history and restoring previous versions; and richer `__like`-based entries search that filters on the collection's title column rather than requiring an exact id match.
